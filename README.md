# Minimal Store UI

Projeto frontend de uma loja fictícia desenvolvido para praticar HTML, CSS e JavaScript.

## Funcionalidades
- Interface de loja simples
- Página de login/registro (UI)
- Tema claro e escuro
- Componentes interativos

## Tecnologias
- HTML5
- CSS3
- JavaScript (Vanilla)

## Objetivo
Projeto criado para fortalecer fundamentos de frontend e portfólio no GitHub.
